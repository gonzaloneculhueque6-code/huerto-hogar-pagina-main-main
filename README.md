# Huerto Hogar — React + Vite (Versión Simple)

Proyecto migrado desde HTML/CSS/JS a React + Vite con enrutamiento básico y carrito simple en `localStorage`.

## Requisitos
- Node.js 18+

## Instalación
```bash
npm install
npm run dev
```

Rutas principales:
- `/` Home
- `/productos`
- `/detalle/:id`
- `/nosotros`
- `/blogs`
- `/contacto`
- `/carrito`
- `/login`
- `/registro`
